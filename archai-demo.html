AUXIO
Latin: aux · voice
Per-object visitor pages for 2,357 collection items — each scannable via QR code, tappable via AUXIO tag, or accessible via hyperlink. No app required. No login. No data collected.

Each page speaks as the object itself: drawing on curatorially-verified metadata, inviting the visitor into a conversation with something that has been waiting in a collection for years, or centuries.
