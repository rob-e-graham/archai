## Documents in This Folder

### ARCHAI_AND_HISTORICAL_DESIGN.md
**Status**: Extensive draft — ready for review

Maps ARCHAI against foundational HCI design principles:
- Shneiderman — Eight Golden Rules and direct manipulation (1982/1985)
- Norman — Affordances, signifiers, conceptual models (1988/2013)
- Dourish — Embodied interaction and phenomenological computing (2001)
- Suchman — Situated action (1987/2007)
- Bill Brown — Thing Theory and object agency (2001–2024)
- Bruno Latour — Actor-Network Theory (1987/2005)
- More-than-human design (Design Museum 2025, DRS 2024, Design+Posthumanism Network 2024)
- Digital repatriation and Indigenous data sovereignty (2024–2025)

For each tradition: original framing, heritage/museum applications, how ARCHAI relates, significance for the PhD.

Key finding: ARCHAI's combination of sovereign infrastructure, semantic interpretation, NFC-bridged physicality, first-person object voice, and institutional data governance does not appear in any system in the 2021–2026 literature surveyed.
