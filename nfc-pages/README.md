# NFC Pages (Fullscreen / HDMI Runtime)

This folder is for dedicated NFC/visitor page templates that will eventually replace inline HTML responses.

Current backend endpoint for quick testing:
- `GET http://localhost:8787/api/nfc/pages/NFC-002`

Planned split:
- `nfc-pages/` static frontend bundle (HTML/CSS/JS)
- `backend-archai` provides JSON page models and chat endpoints
